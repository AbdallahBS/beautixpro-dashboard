## Getting Started

### Installation

```bash
cd beautix-dashboard
npm install
```

### Development

```bash
npm run dev
```

The dashboard will be available at `http://localhost:3001`

### Build for Production

```bash
npm run build
```

## Backend API

Connect to the Beautix backend running on `http://localhost:5000`

## Tech Stack

- React 18
- Vite
- Modern CSS with CSS Variables
- No external UI libraries (pure, lightweight)
